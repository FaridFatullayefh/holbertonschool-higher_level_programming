===============================
How to use 1-my_list.py module
===============================

First, import the class:
    >>> MyList = __import__('1-my_list').MyList

Check if it inherits from list:
    >>> issubclass(MyList, list)
    True

Standard case:
    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

With negative numbers:
    >>> my_list = MyList([1, -4, 2, -3, 5])
    >>> my_list.print_sorted()
    [-4, -3, 1, 2, 5]

With an empty list:
    >>> my_list = MyList([])
    >>> my_list.print_sorted()
    []

With only one element:
    >>> my_list = MyList([10])
    >>> my_list.print_sorted()
    [10]

With non-integer elements (though the task assumes ints, it's good to check floats):
    >>> my_list = MyList([3.5, 2.1, 1.0])
    >>> my_list.print_sorted()
    [1.0, 2.1, 3.5]
