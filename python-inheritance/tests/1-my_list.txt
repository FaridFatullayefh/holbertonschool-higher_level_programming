`MyList` klasını yoxlamaq üçün testlər
=====================================

Klası idxal edirik:
    >>> MyList = __import__('1-my_list').MyList

Yeni bir MyList yaradaq:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Sıralanmış şəkildə çap edək:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Orijinal siyahının dəyişmədiyini yoxlayaq:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Boş siyahı ilə yoxlayaq:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Mənfi ədədlərlə yoxlayaq:
    >>> neg_list = MyList([1, -5, 10, 0])
    >>> neg_list.print_sorted()
    [-5, 0, 1, 10]
